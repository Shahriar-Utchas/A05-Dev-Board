Question-1: What are the different ways to select an element in the DOM?



Elements in the DOM can be selected using different methods. document.getElementById("id") selects an element by its ID, document.getElementsByClassName("class") selects elements by class and returns a collection, document.getElementsByTagName("tag") selects elements by tag name, document.querySelector("selector") selects the first element matching a CSS selector, and document.querySelectorAll("selector") selects all elements matching a CSS selector and returns a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?



innerHTML, innerText, and textContent are used to manipulate content inside an element. innerHTML retrieves or sets the HTML inside an element, including tags. innerText retrieves or sets only the visible text, ignoring hidden elements. textContent retrieves or sets all text content, including hidden elements, but treats everything as plain text, ignoring HTML tags.

Question-3: What is event delegation in the DOM?



Event delegation is a technique where a parent element is used to handle events for its child elements. This is beneficial when dealing with dynamically created elements or optimizing performance. Instead of attaching event listeners to multiple child elements, a single event listener is added to the parent, which detects events from its children.

Question-4: What is event bubbling in the DOM?



Event bubbling occurs when an event triggered on a child element propagates upward through its parent elements. This behavior can cause unintended event triggers on parent elements. To prevent this, event.stopPropagation() can be used to stop the event from bubbling up the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?



Elements in the DOM can be created using document.createElement("tag"). They can be added to the document using parent.appendChild(element). To remove an element, element.remove() or parent.removeChild(element) is used.
